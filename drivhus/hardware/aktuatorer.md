# Lineære aktuatorer — takluker

**HAKIWO 12V lineæraktuator** × 4 (bestilt 2026-05-25, AliExpress)

| Spesifikasjon | Verdi |
|---|---|
| Spenning | 12V DC |
| Slag | 300mm |
| Kraft | 500N |
| Hastighet | 30mm/s (full slag ~10 sek) |
| IP-klasse | IP65 |
| Materiale | Aluminiumslegering |
| Temp.område | -20°C til +75°C |
| Grensebrytere | 2 (innebygd, begge ender) |

## Montering

- Festepunkt: bunn av takluke, 0,70m fra hengsel
- Gir ca. 25° åpningsvinkel fra takplanet
- Motfeste: på takrammen ved siden av luken

## Styring

- 2 reléer per aktuator (H-bro-konfigurasjon)
- Grensebryterne stopper aktuatoren automatisk ved endepunkt
- Aldri aktiver begge reléer samtidig (kortslutning)
