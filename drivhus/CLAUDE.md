# CLAUDE.md — Drivhus

Automatisering av drivhus med sensorer og aktuatorer.

## Mål

- Overvåke temperatur, fuktighet og jordfuktighet
- Automatisk vanning og ventilasjon
- Integrasjon med Home Assistant

## Hardware (planlagt)

- Mikrokontroller: ESP32
- Sensorer: BME280 (temp/fukt), jordfuktighetssensor
- Aktuatorer: relé for pumpe og vifte
- Kommunikasjon: MQTT → Home Assistant
