# CLAUDE.md

Dette repoet inneholder maker- og IoT-prosjekter. Hvert prosjekt har sin egen CLAUDE.md med spesifikk kontekst.

## Struktur

```
prosjekter/
├── drivhus/       ← automatisering av drivhus
├── vaerstasjon/   ← værstasjon
└── modulbygg/     ← modulært byggprosjekt
```

## Konvensjoner

- `README.md` — mål, status, hardware-oversikt
- `logg.md` — arbeidslogg, nyeste øverst
- `todo.md` — oppgaveliste, ferdig flyttes til CHANGELOG.md
- `CHANGELOG.md` — fullførte milepæler
- `hardware/` — pin-mapping, skjemaer, komponentliste
- `kode/` — firmware og skript
